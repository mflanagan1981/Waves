Post-meet recognition report builder — turn the SwimTopia report card into a ready-to-send newsletter.

START HERE How to build this report each week ▼
  1. Pull the report from SwimTopia. Go to Reports → Athlete Report Card ↗ and set the filters exactly like this (these are the settings that make the math line up — same every week):
    Season2026
    Competition CategoryAll
    Minimum / Maximum Ageleave blank
    Show times in course (pool length)S — Short Course Meters (25m)
    Include NT entries in improvementsON
    Do not include point totalsOFF
    Print one page per athleteOFF
    Include results for unofficial meetsON
    Include results for non-public (hidden) meetsOFF
  2. Download the data. Click Download Athlete Report Card Data (CSV) at the bottom of the page. Save it somewhere you can find it.
  3. Drop the CSV into Step 1 below. Everything fills in automatically — the most recent meet is picked for you.
  4. Set the cutoff & review the names (Steps 3–5). Nudge the “seconds dropped” cutoff until the Clock Beaters list feels right (usually 12–18 swimmers), then uncheck anyone you don’t want and pick this week’s biggest drop to feature.
  5. Add the finishing touches (Step 6). Tweak the opening line and add your name. For records set at the meet, pull SwimTopia’s Reports → Records ↗ page and add each one with the quick form (record type, age group, event, name, time) — they aren’t in the report card.
  6. Copy & send. Step 1: click Copy HTML code, then in the SwimTopia email composer ↗ use the </> button to switch to HTML / source view, paste, and click </> again to flip back and preview it before sending. Step 2 (optional): for a PDF to attach or print, click Copy for PDF, paste into a Google Doc, download it as a PDF, and attach that. You’re the one who reviews and sends it — the tool just does the stopwatch math.
